# Scream Bears p5.js

This project is set up to run locally or in the p5.js web editor.

## Files

- `sketch.js`: full game logic, drawing, microphone pitch detection, HUD, start and game-over screens
- `style.css`: page reset for local preview
- `index.html`: local runner with `p5.js` and `p5.sound`

## Use In editor.p5js.org

1. Create a new sketch.
2. Replace the default `sketch.js` with the code from this project's `sketch.js`.
3. Replace `style.css` with this project's `style.css`.
4. In the p5 editor, add the `p5.sound` library if it is not already enabled.
5. Run the sketch and allow microphone access.

## Controls

- Click to enable mic and start
- Hum or sing at a high pitch to jump
- Spacebar or Up Arrow also jumps
- `R` restarts after game over
